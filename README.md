# Cuppie Finance Manager

Микросервисное приложение для управления личными финансами с аутентификацией и учётом транзакций.

## Запуск проекта

В корневой директории выполнить команду:

```bash
docker compose up --build -d
```

Это запустит все сервисы:

* Auth Service
* Finance Backend
* Frontend
* Два экземпляра PostgreSQL

## Настройки баз данных

### База 1 (Finance)

* Порт: `5432`
* Пользователь: `cuppie`
* Пароль: `cupcup`
* База данных: `db`

### База 2 (Auth)

* Порт: `5433`
* Пользователь: `cuppie`
* Пароль: `cupcup`
* База данных: `users_auth_db`

## SQL файлы

В корневой директории находятся два SQL файла, которые создают необходимые таблицы и задают начальные значения для обеих баз данных.
Они автоматически выполняются при первом запуске контейнеров PostgreSQL.
